# RetrospectDiary
회고는 나의 힘. 회고장을 통해 자신을 더 성장시켜 보세요.

### 문제 상황

- 회고를 해야하는데 주기적으로 하지 못한다.
- 다양한 회고 방법중에 어떤 회고방법이 적절한지 모르겠다

### 솔루션

> 회고를 주기적으로, 다양한 방법을 통해 할 수 있는 회고장
> 

### 기능(MVP)

1. 회고 기록 기능
    1. KPT, 5F, 4L, PMI등 다양한 회고의 방법 템플릿 제공
    2. 회고 수정 및 삭제 기능
2. 푸시 알람 기능
    1. 사용자가 회고 주기를 설정해 둘 경우 해당 주기에 push 알람을 보내서 회고를 진행
        1. 시간 설정 기능
        2. 주기 설정 기능
        3. 푸쉬 알림 권한

### 기능(Plus)

1. 회고 공유기능
2. 회고 분석 기능
